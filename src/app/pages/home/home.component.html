Esta es una aplicación web para gestionar proyectos como parte de un proyecto escolar. La aplicación está construida utilizando Angular en el frontend, Firebase para la autenticación y Spring Boot como backend, con una base de datos SQL. El objetivo principal es permitir la creación, asignación y seguimiento de proyectos y tareas entre usuarios.

Características principales:

  • Gestión de usuarios
  • Gestión de pryectos
  • Gestión de tareas
  • Persistencia de datos
  • Interfaz de usuario dinámica

Estado actual

El proyecto ya permite:
  • Autenticación de usuarios.
  • Visualización y creación de proyectos.
  • Gestión básica de tareas asignadas a usuarios.
  • Persistencia de datos en el navegador y comunicación con el backend.
Aúnque podría requerir ajustes en:
  • Optimización del tiempo de carga inicial y gestión de autenticación.
  • Manejo avanzado de errores para evitar interrupciones en la experiencia del usuario.
